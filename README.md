# 애완동물 관리 웹

## 📌 프로젝트 목적
개발 기간: [9월~12월]  담당 역할: [벡 엔드}
반려동물 양육 인구 증가에 따라 체계적인 일정 관리와 건강 관리의 필요성이 대두되면서, 반려동물의 식사, 산책, 병원 방문 등을 효율적으로 관리할 수 있는 웹 애플리케이션을 개발하였습니다.

## 🧠 사용 기술
- JavaScript
- Beact native
- DB : 몽고DB

## ⚙ 구현 내용
- 캘린더 기능
- AI 상담기능
- 애완동물 추가 기능
- 체크리스트
